## STEP 0 · Brand Kit (set up first)
Canva → Brand → Brand Kit. Add:
- **Colors:** Teal `#1E3D3A` · Navy `#16242E` · Dusty blue `#8FA7B3` · Sand/sepia `#D9C9A8` · Cream `#EDE4D3` · Brass `#B0894F`.
- **Fonts:** Headline = the Channel South display serif (or Canva's *Cormorant* / *Playfair Display* as stand-in). Label/wordmark = a clean letter-spaced sans (*Montserrat* works). Accent = a script (*Pinyon Script*) for "Gulfport, Mississippi."
- **Logo:** upload the Channel South wordmark.

---

## TEMPLATE A · FEED POST / CAROUSEL COVER (1080×1350 — 4:5 portrait)
*4:5 fills more feed real estate than square. Build one master, duplicate per treatment.*

**Layout (same on every cover):**
- **Keyline border:** a rectangle inset ~48px from each edge, no fill, 1px stroke in the cover's text color at ~45% opacity. (Editorial frame.)
- **Wordmark:** "CHANNEL SOUTH" top-center, ~60px from top, sans caps, letter-spacing ~4px, ~22px size.
- **Category label:** centered, brass, sans caps, letter-spacing ~3px, ~20px — e.g. PROOF & PROGRESS.
- **Headline:** centered under label, serif, ~78–92px, sentence case, 1–2 lines, generous leading.
- **Footer:** "Gulfport, Mississippi" script ~30px OR a 1px brass rule, centered ~70px from bottom.
- **Margins:** keep all text within the keyline; lots of breathing room.

**Make 4 color variants (duplicate the master, recolor only the background + text):**
1. **Proof / Place** — bg Teal `#1E3D3A` (or full-bleed graded render), text Cream, + a 1px brass **horizon line** across the middle.
2. **Heritage** — bg Sand `#D9C9A8`, text Navy, label brass; add a small "1903" or archival cue.
3. **Hotel Holden** — bg Dusty blue `#8FA7B3`, text Navy.
4. **Credibility / Brand** — bg Navy `#16242E`, text Cream.

## TEMPLATE B · CAROUSEL INTERIOR SLIDE (1080×1350)
- Same keyline + margins as the cover.
- One idea per slide: a short serif line (~56px) top or bottom third, the rest image/render or solid ground.
- Image slides: full-bleed graded render/archival with a subtle navy scrim (20–30% opacity) behind text for legibility.
- No wordmark on interiors (keeps them clean) — or a tiny corner mark only.
- Keep type position identical slide to slide so the swipe feels like one set.

## TEMPLATE C · CTA / END SLIDE (1080×1350)
- Brand ground (Teal or Navy), Cream text.
- Serif: "Follow the rise." + sans: "@channelsouthms" + a brass rule. Same every carousel.

## TEMPLATE D · STORY (1080×1920 — 9:16)
- Same palette + keyline, scaled taller.
- Leave the **top ~250px and bottom ~250px clear** of key text (UI + sticker zones).
- A "swipe up / link" zone at the bottom for the leasing link sticker; a centered zone for poll/quiz stickers.

## STEP · Save as templates + lock
- Group each finished layout → **Save as template** (or duplicate into a "CS — TEMPLATES" folder).
- Lock the wordmark, keyline, and footer elements (select → lock) so only the headline + image swap.
- Name them clearly: `CS-Cover-Proof`, `CS-Cover-Heritage`, `CS-Cover-Holden`, `CS-Cover-Brand`, `CS-Interior`, `CS-CTA`, `CS-Story`.

## GRID-RHYTHM CHECK (before posting)
Lay the next ~6 covers in Canva side by side: **alternate dark (Teal/Navy) and light (Sand/Dusty) tiles** — never 3 same-tone in a row. That's what makes the profile read as one intentional brand.

## DESIGN GUARDRAILS
- Color-grade every render/photo warm + sunlit (Canva: Adjust → warm temp, lift shadows) so all imagery sits in the palette.
- Sentence case everywhere except the wordmark + labels (caps).
- Two type weights max. Lots of white space. No drop shadows, no busy overlays — restraint is the brand.
