# Channel South — Cover & Grid System
*Evie · 2026-06-15 · The rules that make every post + carousel cover look like one intentional brand. Build these as locked Canva templates so it's repeatable, not redesigned each time.*

> Goal: when someone lands on @channelsouthms, the grid should read as **one considered brand** (Alys-Beach-level restraint), and each carousel should feel like a designed set, not loose slides. Consistency lives in the **cover system + the palette + the grade.**

---

## 1 · THE PALETTE (lock these — no others)
| Use | Color | Hex |
|---|---|---|
| Primary brand (deep coastal) | Teal/slate | `#1E3D3A` |
| Deep ground | Navy | `#16242E` |
| Hotel Holden | Dusty blue | `#8FA7B3` |
| Heritage ground | Warm sand/sepia | `#D9C9A8` |
| Light text / cream | Cream | `#EDE4D3` |
| Accent | Brass | `#B0894F` |
*Every render and photo gets color-graded warm + sunlit so it sits in this family. One grade across everything = instant cohesion.*

## 2 · TYPE
- **Headlines:** elegant serif (the Channel South display serif). Sentence case. 2 lines max on a cover.
- **Category label:** small caps, letter-spaced ~2.5px, brass or muted — names the pillar (PROOF & PROGRESS · THEN & NOW · HOTEL HOLDEN · THE DISTRICT · GULFPORT, DID YOU KNOW?).
- **Wordmark:** "CHANNEL SOUTH" letter-spaced caps, small, **same placement every time** (top-center).
- **Accent/footer:** script "Gulfport, Mississippi" or a brass keyline. Two weights only; never bold body.

## 3 · THE COVER ANATOMY (every cover, same skeleton)
1. **Wordmark** top-center (micro, letter-spaced).
2. **Thin keyline border** inset ~9px (editorial frame — the restraint cue).
3. **Category label** (brass small caps) — tells the viewer the series.
4. **Serif headline** centered, 1–2 lines.
5. **Footer** accent (script locale or brass rule).
*Only the background treatment changes by type — the skeleton never does.*

## 4 · COVER TREATMENTS BY TYPE (the grid rhythm)
- **Proof / Place (render covers):** deep teal or navy ground (or full-bleed graded render) + cream text + a thin brass **horizon line**. Cinematic.
- **Heritage (Then & Now, fun facts, Captain Jones):** warm sand/sepia ground + navy text + an archival cue ("1903"). Reads instantly as the history series.
- **Hotel Holden reveals:** dusty-blue brand ground + navy text + keyline. Its own recognizable sub-look.
- **Credibility / Brand:** navy ground + cream serif.
**Rhythm rule:** alternate dark (render/navy) ↔ light (sepia/dusty) tiles down the grid so the profile has a calm checkerboard rhythm — never three identical-tone tiles in a row.

## 5 · CAROUSEL INTERNAL CONSISTENCY
- **Slide 1 = the cover** (full anatomy above).
- **Interior slides:** same margins, same type hierarchy, one idea per slide, lots of white space; archival full-bleed or render full-bleed with a consistent text plate.
- **Final slide = CTA**, always the same lockup: "Follow the rise. @channelsouthms" on the brand ground.
- **Swipe cue** on slide 1 (a small brass arrow / "swipe") so people know to swipe.
- Keep every carousel to **6–7 slides**.

## 6 · BUILD IT ONCE (Canva)
Make a **locked template set**: one cover template per treatment (Proof, Heritage, Holden, District/Brand) + one interior-slide template + one CTA-slide template. Lock position/type/color; only swap the headline + image. Save as a Brand Kit (palette + the two fonts + the wordmark). Then every future post is fill-in-the-blank and on-system by default.

## 7 · QUICK CHECKLIST (before anything posts)
- [ ] Wordmark in the same spot? - [ ] Category label present + correct? - [ ] Serif headline, sentence case, ≤2 lines? - [ ] Palette colors only? - [ ] Warm coastal grade applied? - [ ] Keyline/footer consistent? - [ ] Does it alternate tone with the last 2 grid tiles?
