System Setup

1. Claude Code - Automation Schedules

These run automatically in the background and update your system.

Monday 8:00 AM - News Monitoring
/schedule create "news-monitoring" \ --run "Monday 8:00 AM" \ --prompt "Generate 2-min news briefs for all 5 clients: - Channel South: hotels, local Gulfport news - Bellamare: real estate, market updates - Gallery Madison: art events, Madison news - Germantown Village: retail, community events - Marée Creative: creative industry trends Save to ~/04_Research/weekly-briefs/"
Every Monday 10:00 AM - Analytics Parser
/schedule create "analytics-parser" \ --run "Every Monday 10:00 AM" \ --prompt "Read all CSVs from ~/03_Analytics/meta-exports/ Extract: followers, engagement rate, top posts, reach Generate summary: ~/03_Analytics/performance-reports/ Update equity-tracking.md with monthly progress"
Every Friday 5:00 PM - Weekly Reports
/schedule create "weekly-reports" \ --run "Every Friday 5:00 PM" \ --prompt "Generate client reports for all 5 accounts: - Followers this week - Engagement rate - Top performing post - What we accomplished - What's coming next Ready to send to clients"
Every Day 6:00 AM - Daily Checklist
/schedule create "daily-checklist" \ --run "Every day 6:00 AM" \ --prompt "Generate today's checklist from ~/02_Systems/weekly-checklist.md Show: - Engagement window time (9:30-10 AM) - Posts going live today - Any client updates - What needs attention"

2. Cowork Hooks - Triggers & Notifications

These trigger actions and send you notifications at specific times/events.

9:30 AM Daily - Engagement Window Alert
Hook: Every morning at 9:30 AM Action: Send notification Message: "Engagement Window Now (30 min) - Check all 5 accounts - Reply to top DMs - Monitor comments"
Monday 9:00 AM - Weekly Checklist
Hook: Every Monday 9:00 AM Action: Send notification + file Message: "Week's Batching Plan Ready - Mon: Channel South (90 min) - Tue: Bellamare (90 min) - Wed: Gallery Madison + Germantown (180 min) - Thu-Fri: Post + Engage + Report" Attach: ~/02_Systems/weekly-checklist.md
Post Goes Live - Auto-Log Performance
Hook: When post is published on Meta Action: Auto-log to performance tracker Track: Reach, Engagement, Saves, Shares, Comments Update: ~/03_Analytics/[client]-week-performance.json
Friday 5:00 PM - Report Ready
Hook: Every Friday 5:00 PM Action: Generate + send notification Message: "Weekly Reports Ready" Files: - ~/03_Analytics/performance-reports/week-X-summary.pdf - ~/03_Analytics/equity-tracking.md (your equity log) - Ready to email to clients

4. Monthly Workflow

Monday: News briefs auto-generate → Review → Plan week's batching
Mon-Wed: Batch content for all 5 clients in calendar files
Thu-Fri: Upload assets, post, engage (9:30-10 AM window)
Next Monday: Upload Meta CSVs → Analytics auto-parse → Performance snapshot → Repeat
